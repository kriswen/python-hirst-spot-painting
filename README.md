## How to use 

   ```bash
# Draw the painting with the following parameters:
# - n: Number of dots horizontally
# - m: Number of dots vertically
# - path: Spacing between dots
# - dot_size: Size of each dot
draw_painting(10, 10, 50, 20)
   ```
